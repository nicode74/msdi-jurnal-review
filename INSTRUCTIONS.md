# Project Context: Journal Paper Review Task
Objective: Extract core insights from the attached Markdown paper to create a formal report and presentation slides.

## 1. Extraction Rules:

    Priority 1: Identifikasi masalah utama (Research Gap).

    Priority 2: Bedah metodologi secara teknis (Step-by-step).

    Priority 3: Temukan hasil utama dan perbandingannya dengan penelitian sebelumnya.

    Priority 4: Kritik terhadap paper (Kekurangan/Limitas).

## 2. Output Specifications:

    For Report: Gunakan bahasa Indonesia formal, struktur akademik (Pendahuluan, Metodologi, Hasil, Analisis).

    For Presentation: Buat poin-poin singkat (Max 5 poin per slide) dengan alur: Hook -> Problem -> Method -> Results -> Conclusion.

## 3. Tone & Style:

    Gunakan terminologi Informatika yang akurat.

    Jangan berbasa-basi. Langsung ke data teknis.
